# 💰 Expense Tracker

A full-stack Expense Tracker web application built using the MERN Stack that helps users manage their expenses, visualize spending patterns, and gain AI-powered financial insights.

## 🚀 Live Demo

🔗 Frontend:(https://expense-tracker-frontend-thf0.onrender.com)

## 📂 GitHub Repository

🔗 (https://github.com/10AP03/Expense-Tracker)

---

## 📌 Features

- 🔐 Secure User Authentication using JWT
- ➕ Add New Expenses
- ✏️ Edit Existing Expenses
- ❌ Delete Expenses
- 📊 Interactive Dashboard
- 📈 Expense Analytics & Charts
- 🏷️ Expense Categorization
- 👤 User Profile Management
- 🤖 AI-Powered Financial Assistant
- 📱 Responsive User Interface

## 📖 Application Workflow

1. Register a new account.
2. Login securely using JWT Authentication.
3. Add, edit, or delete expenses.
4. View spending history.
5. Analyze expenses using charts.
6. Get AI-powered financial insights.
7. Manage your profile.
